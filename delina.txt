langkah-langkah mengupload file/berkas ke dalam repository :
1.Masuk ke akun Git.
2.Buat repository yang bersifat public dengan nama “UAS_RPL”
3.Lakukan clone repository kosong tersebut ke komputer local, dengan mengetikkan perintah berikut ;
	$ git clone https://github.com/Delina1406029/UAS_RPL.git
4.Buat sebuah berkas baru dengan format bebas (delina.txt).
5.Lakukan commit dan push perubahaan ke repository di GitHub,

	$ git add delina.txt
  Kedua, melakukkan commit untuk mengirim pesan dengan perintah berikut :
	$ git commit –m “pesan”
  Terakhir, kirim file ke repository. Ketikan perintah berikut : 
	$ git push : memasukan username dan password GitHub anda.
